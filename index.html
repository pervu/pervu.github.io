ESP32 Arduino library for AS5600 and AS5600L Magnetic Encoder
Work with signals from I2C bus, work with signals from analog(PWM) pin, configure all AS5600 registers, get the Raw Angle, get the Deg Angle, get the Rotation Speed, check the magnet, get magnetic intensity, filter input signals
Git link

Encoder modes modetype_t
I2C - encoder operation via i2c bus
I2CPWM - configuration via i2c bus, receiving data as an analog (PWM) signal
PWM - receiving data in the form of an analog signal, in this mode it is impossible to write to AS5600 registers

Two ES5600 update modes:
timer interrupts for ESP32;
loop updates for arduino, esp8266 or ESP32;

Read more on the AS5600 github page.
